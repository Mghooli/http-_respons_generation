# 🐶 HTTP Dog Code Manager

A simple web app to search, filter, save, and manage lists of HTTP response codes with fun dog images from [https://http.dog](https://http.dog).

## ✨ Features

- 🔐 User login/signup (each user has personal lists)
- 🔍 Filter HTTP codes by:
  - Exact (e.g., 404)
  - Patterns (e.g., 2xx, 20x)
- 🐶 View matching dog images
- 💾 Save filtered lists with custom names
- 📋 View, edit, and delete saved lists

## 🛠 Tech Stack

- **Frontend**: HTML, CSS, JS  
- **Backend**: Java Servlets  
- **Database**: MySQL  
- **Images**: [http.dog](https://http.dog)  

## 📄 Pages

1. Login/Signup  
2. Search and Filter  
3. List Management  

## ☁️ Deployment

Hosted on a free platform like Render or Railway.
